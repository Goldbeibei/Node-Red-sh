# 歡迎使用JF整理nodered一件安裝腳本

**目前測試適用於ubuntu22.04LTS**

## 使用教學
1. 更新ubuntu並安裝curl
   `sudo apt-get update`
   `sudo apt install curl`


2. 下載腳本

   `curl -JLO https://github.com/Goldbeibei/Node-Red-sh/blob/main/install_nodered.txt`

3. 賦予腳本權限並執行

   `sudo chmod 775 install_nodered.sh`
   `./install_nodered.sh`
